Willkommen zur Produktivitäts-Studie!

In diesem Test werden Ihnen nacheinander verschiedene Buchstaben oder Zahlen auf dem Bildschirm angezeigt. Ihre Aufgabe ist es, so schnell und genau wie möglich auf bestimmte Zeichen mit der Leertaste zu reagieren.

Manchmal sollen Sie bei einem bestimmten Zeichen drücken, manchmal bei allen außer einem bestimmten Zeichen. Vor jedem Block erhalten Sie eine kurze Erklärung, welche Regel gerade gilt.

Falls eine Unterbrechung startet, klicken Sie bitte auf den Pause-Button. Der Test kann erst fortgeführt werden, wenn die Unterbrechung beendet ist und Sie den Button wieder auf Fortsetzen gestellt haben.

Der gesamte Test dauert einige Minuten. Bitte konzentrieren Sie sich gut.

Wenn Sie bereit sind, klicken Sie auf „Start“, um den Test zu beginnen.
